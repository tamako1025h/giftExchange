## giftExchange
個人の勉強の一環でつくりました。プレゼント交換する相手を抽選するプログラム
## 使い方
1. pipでインストール
```
pip install git+https://github.com/tamako1025h/giftExchange
```
2. 動かす
```
giftExchange
```
3. アンインストール
```
pip uninstall giftExchange
```
